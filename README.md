# テトリス (HTML版)

## 概要

このプロジェクトは、HTML, CSS, そしてVanilla JavaScriptのみで作成されたクラシックなテトリス風ゲームです。サイバーパンク風のレトロフューチャーなデザインと、多彩なBGM・効果音システムが特徴です。

## 遊び方（操作方法）

- **z**: ブロックを左に移動
- **c**: ブロックを右に移動
- **x**: ブロックを下にドロップ
- **s**: ブロックを回転
- **Enter**: ゲームの一時停止 / 再開

### メニュー画面

- **↑ / ↓**: ボタンの選択
- **Enter**: 決定

## 工夫した点（実装した機能）

### 1. 動的なBGMシステム
- **ランダム再生:** ゲームをプレイするたびに、3曲のBGMの中からランダムで1曲が選ばれ、毎回新鮮な気持ちで楽しめます。
- **シーン別BGM:** スタート画面、ゲームオーバー画面でもそれぞれ専用のBGMが流れ、雰囲気を盛り上げます。
- **フェードイン:** 各画面に遷移する際、BGMが自然にフェードインする演出を加えています。

### 2. 多彩な効果音
ブロックの移動、回転、ドロップ、ライン消去、UIのボタン操作など、様々なアクションに効果音が設定されており、操作の感触をリッチにしています。

### 3. 洗練されたUI/UX
- **デザイン:** 全体的に「サイバーパンク」をテーマとし、ネオンが光るようなデザインやアニメーション背景を採用しました。
- **オーバーレイ:** スタート、ポーズ、ゲームオーバー画面には、すりガラスのような半透明のオーバーレイ（グラスモーフィズム）を使用し、モダンな見た目に仕上げています。

### 4. ゲームオーバー画面
- **ハイスコア表示:** 上位3位までのハイスコアが記録・表示されます。
- **新記録のハイライト:** プレイ結果がハイスコアにランクインした場合、「← You」というインジケーターが点滅し、プレイヤーに分かりやすく通知します。

### 5. 完全なキーボード操作
マウスを使わずに、スタートメニューからゲームプレイ、ゲームオーバー後の選択まで、すべての操作をキーボードのみで完結できます。
